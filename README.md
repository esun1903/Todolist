# Todolist
1학기 관통프로젝트 👍📝✏️


1차

| 메소드 | 경로        | 설명                 | 인증 |
| ------ | ----------- | --------------------|---- |
| GET    | /todo       | 모든  todolist 조회  | O    |
| GET    | /todo/{Idx} | todoLIst 하나씩 조회 | O    |
| POST   | /todo       | todolist 생성        | O    |
| PUT    | /todo       | todolist 수정하기    | O    |
| DELETE | /todo/{Idx} | todolist 삭제하기    | O    |
| GET    | /           | 모든  todolist 조회  | O    |
